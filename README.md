# 🌱 AgroTech Connect

Sistema fullstack para gerenciamento de startups do agronegócio.

## 📚 Tecnologias utilizadas
- Node.js
- Express
- React
- Vite

## ⚙️ Funcionalidades
- Listar startups
- Cadastrar novas startups
- Editar startups
- Excluir startups

## 🔗 API (Backend)
Rodando em:
http://localhost:3000/listaStartups

## 💻 Frontend
Rodando em:
http://localhost:5173
